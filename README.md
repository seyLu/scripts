# Collection of Scripts

### Usage: Bash Shell Scripts

1. Append on `~/.bash_aliases`

    ```bash
    # Initialize External Scripts
    [[ -z "${_SCRIPTS_PATH}" ]] && source "${HOME}<path_to_scripts>/scripts/bash/__init__"
    ```
2. On Bash Shell, run
    
    ```bash
    $ build
    ```

3. Enjoy
